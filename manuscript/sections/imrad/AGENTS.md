# IMRAD Fragment Notes

This tree contains editable manuscript fragments. The numbered files under
`../../` are composed outputs for matrix sections; edit the sources here instead.

- Section bindings and output names live in `../../sheaf/manifest.yaml`.
- Fragment track types and generated renderers live in `../../sheaf/tracks.yaml`.
- Quantitative claims must use hydrated `{{token}}` values or cite generated
  artifacts validated by `validate_outputs` / `validate_manuscript`.
